Kimi Models for GitHub Copilot Chat uses your own Kimi (Moonshot AI) API key to make Kimi models available in the Copilot model picker.

Your key is stored in VS Code's SecretStorage (the OS keychain). It is never written to `settings.json` or your Git history.

Paste it once, then update or remove it later from the Command Palette.

- `Cmd/Ctrl + Shift + P`: Open the Command Palette
- `Kimi: Set API Key`: Set or update your API key
- `Kimi: Clear API Key`: Remove your API key
- `Kimi: Get API Key`: Open the API-key page for your selected region
